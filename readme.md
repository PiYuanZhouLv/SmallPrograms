## 3388：暴破24点

起因是一个同学问3 3 8 8怎么求24点，于是写了一个brute24.py

原理是将所有可能计算树列出并验证（实际上是枚举计算树的后序排列）

得出唯一解： $ \frac{8}{3-\frac{8}{3}} = 24 $
